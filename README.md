## dependencies
- pytorch (torch)
- pytorch vision (torchvision)
- python-chess (chess)

install automatically:
```
pip install -r requirements.txt
```
install manually:
```
pip install torch
pip install torchvision
pip install chess
```
## run
```
python main.py
```
